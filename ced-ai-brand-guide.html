Usage Guidelines

Principles for consistent application

✓ Do

  • Use Complementary as the default for all primary touchpoints
  • Reserve Triadic/Tetradic for content that earns the energy
  • Keep email correspondence in Monochromatic only
  • Pair EB Garamond with DM Sans for clear hierarchy
  • Use JetBrains Mono for any code or technical data
  • Use EB Garamond for long-form reading

✗ Don't

  • Mix colors across harmony groups (no Triadic yellow with Complementary orange)
  • Use Tetradic for routine content — save it for launches
  • Put orange or other accents in email signatures
  • Use EB Garamond for short UI text — it's for display and long-form
  • Combine more than two font families in one context
  • Override the semantic system (error is always red, etc.)

Context Triggers

  • Complementary: Website, apps, social, presentations
  • Monochromatic: Email, formal letters, invoices
  • Triadic: Blog posts, announcements, featured work
  • Tetradic: Product launches, demos, experimental features

Font Loading

  • Load via Google Fonts or self-host for performance
  • EB Garamond: 400, 500, 600, 700 + italics
  • DM Sans: 400, 500, 600, 700
  • JetBrains Mono: 400, 500, 600
